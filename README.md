# CS614 Final Project
## AI Industrial Quality Inspector

### Overview
This project explores computer vision methods for automated industrial surface defect detection using transfer learning and explainable AI.

### Objective
Build a multi-class image classification system that detects manufacturing defects and provides visual explanations using Grad-CAM.

### Technical Approach
- Transfer learning using pretrained ResNet18
- Multi-class classification
- Explainable AI (Grad-CAM visualization)

### Project Structure
data/        -> datasets (raw and processed)
notebooks/   -> experimentation and analysis
src/         -> reusable training and model code
outputs/     -> trained models and generated figures
